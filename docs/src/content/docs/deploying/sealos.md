---
title: Sealos
description: Deploying with SeaPack on Sealos
---

[Sealos](https://sealos.com) is the target platform for SeaPack.

SeaPack is designed to fit into a Sealos build pipeline. It analyzes the source
tree, selects a supported language provider, generates a BuildKit plan, and
produces a runnable image that can be deployed by Sealos.
