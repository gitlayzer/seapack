# Rails Starter Example

This is a [Ruby on Rails](https://rubyonrails.org/) starter app that connects to a Sealos Postgres database and supports [Action Cable](https://guides.rubyonrails.org/action_cable_overview.html).

[![Deploy on Sealos](https://sealos.app/button.svg)](https://sealos.app/new/template/sibk1f)

## ✨ Features

- Ruby
- Rails
- Postgres
- Redis

## 💁‍♀️ How to use

- [Create a Sealos project with the Postgres plugin](https://sealos.app/project?plugins=postgresql)
- Connect to your Sealos project with `sealos link`
- Install Ruby requirements `bundle install`
- Migrate the database `sealos run rake db:migrate`
- Run Rails `sealos run bin/rails server`

## 📝 Notes

This app was generated with the `rails new` command. Read more about Rails on
their [official website](https://rubyonrails.org/)
